# CLI API
:::subtitle
Reference for CLI
:::

The xyd CLI is one of the primary ways to interact with the xyd platform. You’ll use the `xyd` command to run your docs locally, build them for production, and manage your project.

First, install xyd:
:::code-group{}
```bash bun
bun add -g xyd-js
```

```bash pnpm
pnpm add -g xyd-js
```

```bash npm
npm i -g xyd-js
```
:::

Then run `xyd` in your project to start the dev server, or `xyd build` to produce a static site. Each command’s full reference is listed in the sidebar.
